========================
STRICT RULES
============

1. FOOTING / COLUMN IDENTIFICATION

A) If the table header is a footing label, such as:
RAFT-1, RAFT-2, R1, F1, BF1, CP1, CF1, RW, AF1, BRF2, AC1, NF1, FC1, ARF1, FP1, CWF1, CWF2A

Then:

* footing_id = exact header value
* column_id = null unless a column number is explicitly provided

B) If the table header is a column number, such as:
C1, C2, C12, C25

Then:

* column_id = exact column header value
* footing_id must be taken only from the footing designation/reference shown in that same column or mapped schedule cell

Do not assume:

* C1 = F1
* C12 = F12
* Column number automatically equals footing number

Only use explicit schedule mapping.

If footing_id is unclear, return null.

2. COLUMN LOCK

For each footing or column header:
Only read values vertically under that exact header.
Never shift left or right.
Never borrow values from neighboring columns.
Never merge values from adjacent footing columns.

3. IGNORE COMPLETELY

Ignore rows labeled:

* B' x L' (PCC)
* PCC SIZE
* PCC
* LEAN CONCRETE SIZE

Also ignore:

* T row
* REMARK row
* Any other row not explicitly required below

4. SIZE EXTRACTION

From row labeled:
B x L
B × L
WIDTH x LENGTH
SIZE
FOOTING SIZE

Extract:

* width from B / width value
* length from L / length value

If the cell contains:
REFER PLAN
AS PER PLAN
SEE PLAN
--------

Then:

* width = null
* length = null

From row labeled:
D
DEPTH
THK
THICKNESS

Extract:

* depth = value

If unclear or unavailable:

* depth = null

5. REINFORCEMENT

Extract only from reinforcement rows, including:

STEEL // - B (SHORT SIDE) (BOTTOM FACE)
STEEL // - L (LONG SIDE) (BOTTOM FACE)
STEEL // - B (SHORT SIDE) (TOP FACE)
STEEL // - L (LONG SIDE) (TOP FACE)
BOTTOM STEEL
TOP STEEL
SHORT SIDE STEEL
LONG SIDE STEEL
MAIN BAR
DISTRIBUTION BAR
REINFORCEMENT

These are vertical detail rows in the schedule. The bar-and-spacing values
inside these rows must go into reinforcement.dia and reinforcement.spacing
for the footing/header directly above that same column.

For each cropped footing/header strip, extract all visible reinforcement
details:

* bar diameter exactly as written, including repeated occurrences when the same bar appears in multiple steel rows
* spacing for each visible bar value
* stirrup/tie diameter and stirrup/tie spacing if a stirrup, links, tie, or shear reinforcement row/note is visible
* concrete mix if a MIX / CONCRETE GRADE / GRADE OF CONCRETE value is visible
* grade of steel if a STEEL GRADE / REBAR GRADE / GRADE OF STEEL value is visible

If stirrups/ties are visible:

* stirrups.dia = visible stirrup/tie diameter values
* stirrups.spacing = visible stirrup/tie spacing values

If cell contains:
16T-100 c/c
16T @ 100 c/c
T16 @ 100 C/C
12T-150

Extract:

* dia = "16T", "T16", "12T", etc.
* spacing = "100 C/C", "150 C/C", etc.

Rules:

* Keep dia exactly as written.
* Do NOT convert dia format.
* Convert spacing to uppercase format: "100 C/C"
* If spacing is missing, include dia only if clearly visible.
* If cell contains "-", blank, or unreadable text, ignore it.
* Do NOT merge reinforcement values from other columns.
* Do NOT guess.
