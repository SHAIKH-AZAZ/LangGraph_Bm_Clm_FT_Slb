=========================
PATTERN-3 TABLE STRUCTURE
=========================

This is a tabular beam schedule with the following columns (left to right):

  BEAM MARKED  |  SIZE (B x D)  |  REINFORCEMENT columns  |  STIRRUPS columns  |  REMARKS

REINFORCEMENT columns:
  - BOTTOM (STRAIGHT)
  - BOTTOM (CURTAIL)
  - TOP (STRAIGHT)
  - TOP (EXTRA OVER SUPPORT) LEFT
  - TOP (EXTRA OVER SUPPORT) RIGHT

STIRRUPS columns:
  - STIRRUPS (UPTO L/4)
  - STIRRUPS (REST)

=========================
MULTIPLE TABLES PER PAGE
=========================

A single page may contain MULTIPLE separate beam schedule tables
(e.g. BLOCK 1A ROOF, BLOCK 1A MIDLEVEL, BLOCK 1B ROOF, BLOCK 1B MIDLEVEL) or any other type may be given but table format is must to convsider .

Extract ALL beams from ALL tables on the page.
Do NOT skip any table section.

=========================
BEAM ID RULES  ← READ CAREFULLY
=========================

Extract beam_id EXACTLY as written in the BEAM MARKED column.
Copy the text character-for-character, including all commas, numbers, and letters.

CRITICAL — GROUPED IDs:
A single cell often contains multiple beam numbers joined by commas, e.g.:
  AB3,4,8,9,40,41,44,45
  AB52,59,99,100
  AB53,54,55,56,57,58,
  BB2,39,43

These are ONE beam_id string — keep the ENTIRE cell text as one string.
A trailing comma (e.g. "AB53,54,55,56,57,58,") is part of the label — keep it.

Do NOT split grouped IDs into separate beam entries.
Do NOT treat the numbers after commas as reinforcement quantities.
Do NOT treat the numbers after commas as beam sizes.

VALID beam_id examples from this drawing type:
  AB2,39,43          AB3,4,8,9,40,41,44,45      AB7
  AB52,59,99,100     AB12,34                     AB20,27
  AB53,54,55,56,57,58,   AB71,72,73
  BB2,39,43          BB3,4,8,9,40,41,44,45
  RB1, RB2                    B1, B2

=========================
WHAT TO IGNORE
=========================

The image may contain elements that are NOT part of the beam schedule table:

1. FLOOR PLAN / STRUCTURAL GRID on the left side of the image
   - Grid labels like BC41, BC43, BB39, BC44 are structural grid references
   - These appear IN THE FLOOR PLAN, not in the table
   - DO NOT extract them as beam IDs

2. REMARKS column (rightmost column)
   - May contain text like "ROOF LVL BEAM", "MIDLEVEL BEAM", "CANTILEVER"
   - IGNORE the REMARKS column entirely
   - Do not include remarks text in beam_id, reinforcement, or stirrups

3. TABLE TITLE rows
   - e.g. "SCHEDULE OF BEAMS AT ROOF FLOOR LEVEL :- BLOCK 1A"
   - These are headings, not data rows — ignore them

4. EMPTY ROWS at the bottom of a table — skip them

=========================
SIZE RULES
=========================

1. Extract width and depth from the SIZE (B x D) column.
2. Remove units — extract numbers only.
   Example: "200 x 600" → width: 200, depth: 600
3. length = null (no span column in this table).

=========================
REINFORCEMENT RULES
=========================

1. Collect values from ALL 5 reinforcement columns:
   - BOTTOM (STRAIGHT)
   - BOTTOM (CURTAIL)
   - TOP (STRAIGHT)
   - TOP (EXTRA OVER SUPPORT) LEFT
   - TOP (EXTRA OVER SUPPORT) RIGHT

2. Skip any cell that contains only "---" or is blank.

3. Normalize all values to format:  quantity-Tdiameter
   Examples:
     2-T16  → 2-T16   (already correct)
     2 T16  → 2-T16
     3T12   → 3-T12
     2-12   → 2-T12

4. If "+" appears in a cell, split into separate entries:
   "2-T16 + 2-T12" → "2-T16" and "2-T12"

5. Remove duplicates within the same beam.
6. Do NOT mix reinforcement from different rows.

=========================
STIRRUPS RULES
=========================

Stirrups are written as:  T8@150C/C  or  T8@200 C/C  or  T8@150C/C

Parse each stirrup value as follows:
  - dia: the T-bar part before "@"  → e.g. "T8"
  - spacing: the number after "@" before "C/C" → format as "NNN C/C"

Examples:
  T8@150C/C  → dia: "T8",  spacing: "150 C/C"
  T8@200C/C  → dia: "T8",  spacing: "200 C/C"
  T8@150 C/C → dia: "T8",  spacing: "150 C/C"

Rules:
1. Two stirrup columns exist: UPTO L/4 and REST.
   Extract both — add each unique spacing to stirrups.spacing[].

2. Add the dia to stirrups.dia[] (deduplicate — usually "T8" appears once).

3. Skip cells that are "---" or blank.

4. Do NOT include stirrup text in the reinforcement[] array.
5. Do NOT include remarks text (CANTILEVER, ROOF LVL BEAM) in stirrups.

Example result for "T8@150C/C" (UPTO L/4) and "T8@200C/C" (REST):
  stirrups.dia:     ["T8"]
  stirrups.spacing: ["150 C/C", "200 C/C"]

If both columns have the same spacing, include it only ONCE.

=========================
GENERAL RULES

1. Return ONLY valid JSON — no explanation, no markdown.
2. Do NOT guess missing values.
3. If a cell is blank or "---" → skip it (do not add null or empty string).
4. Extract ALL rows from ALL tables visible in the image.
5. Each data row in the table = one beam entry in the JSON.
