RULE 1 — BEAM ID
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
The label is the LOWEST text beneath a cross-section rectangle.
Extract beam_id EXACTLY as written — including all prefix letters,
floor codes, underscores, hyphens, and letter suffixes.

STANDARD:
  B1  B2  B2a  B2b  P1
  → beam_id = "B1", "B2", "B2a", "P1"

TYPE-PREFIX (keep full prefix, 1–4 letters before number):
  Lintel/Roof  : LB1  LB1A  LB1a  RB1
  Cantilever   : CB1a
  Drop/Deep    : DB1a
  Transfer     : TB1a  TB1
  Plinth       : PB1
  Stair        : SB1  SBT1
  Main/Misc    : MB1  AB1  BB1  HB  RR
  Band/Slab    : BS1  BS2a  (may be lowercase: bs1  bs2a)
  Special      : FFB12  TFB1  PTBB1  LBK1  LBK12
  → beam_id = "LB1A", "CB1a", "PB1", "BS1", "FFB12" …

FLOOR-PREFIXED (prefix_code format — keep the ENTIRE string):
  1F_B47(200x450)  → beam_id = "1F_B47"
  MF_B8            → beam_id = "MF_B8"
  LGF_B53(300x550) → beam_id = "LGF_B53"
  UGF_B97a(450)    → beam_id = "UGF_B97a"
  SGF_B43(450x600) → beam_id = "SGF_B43"
  STB_B2(300x450)  → beam_id = "STB_B2"
  Common floor codes: 1F  2F  MF  GF  UGF  LGF  SGF  STB  LPTB  …
  Rule: anything before the last underscore is a floor/zone code — KEEP it.

HYPHEN — two different roles, must distinguish:
  • Hyphen before WxH size → SIZE SEPARATOR, strip it and everything after:
      B58d-600X600  → beam_id = "B58d"
  • Hyphen before a SINGLE number → PART OF THE ID, keep it:
      LPTB-5(900x550)  → beam_id = "LPTB-5"

SIZE IN LABEL — strip after "(" or after a size-separator hyphen:
  PB1(300x600)     → beam_id = "PB1"       size 300×600
  UGF_B97a(450)    → beam_id = "UGF_B97a"  size width=450, length=null
  LPTB-5(900x550)  → beam_id = "LPTB-5"    size 900×550

GROUPED BEAMS — one cross-section carries two IDs in parentheses:
  (B1,B2)  or  B1+B2
  → extract as TWO separate beams, each with the SAME reinforcement/stirrups.
  → beam_id = "B1"  and  beam_id = "B2"

COMBINED ID with +:
  LBK1+LBK12  → beam_id = "LBK1+LBK12"  (keep as-is, no split)
  Rule: combined IDs have NO space around "+" and mix non-identical strings.

CASE — IDs can be all-caps, lowercase or mixed. Keep exactly as written:
  BS1  bs1  Bs2a  UGF_B97a  1F_B47  LPTB-5  all valid.

MUST-DO:
  ✓ Include floor code prefix (1F_, LGF_, STB_ …) in beam_id.
  ✓ Include type prefix (LB, RB, CB, PB, SB …) in beam_id.
  ✓ Include letter suffix (a, b, A …) in beam_id.
  ✓ Strip only the SIZE part after "(" or size-separator hyphen.
  ✗ NEVER include dimension numbers in beam_id.

REAL-WORLD LABEL EXAMPLES — all of these are valid beam IDs:
  Plain / Continuous  : B1   B2a   P1
  Lintel / Roof       : LB1  LB1A  LB1a  RB1  RB1(300x600)  → beam_id="RB1"
  Cantilever          : CB1a
  Drop / Deep         : DB1a
  Transfer            : TB1a  TB1
  Post-tensioned/Band : PTBB1
  Plinth              : PB1  PB1(300x600)  → beam_id="PB1"
  Stair               : SB1  SBT1
  Main / Misc         : MB1  AB1  BB1  HB  RR  FFB12  TFB1
  Band / Slab strip   : BS1  bs1  bs2a  (case preserved)
  Lintel combined     : LBK1+LBK12  → keep as-is
  With size in label  : P1(300x600)      → beam_id="P1",    size 300×600
                        PB1(300x600)     → beam_id="PB1",   size 300×600
                        RB1(300x600)     → beam_id="RB1",   size 300×600
                        LB1(300x600)     → beam_id="LB1",   size 300×600
                        TB1(300x600)     → beam_id="TB1",   size 300×600
  Grouped on one rect : (B1,B2) → TWO beams "B1" and "B2", same bars
  Floor-prefixed      : 1F_B47(200x450)  → beam_id="1F_B47"
                        MF_B8            → beam_id="MF_B8"
                        LGF_B53(300x550) → beam_id="LGF_B53"
                        UGF_B97a(450)    → beam_id="UGF_B97a"
                        SGF_B43(450x600) → beam_id="SGF_B43"
                        STB_B2(300x450)  → beam_id="STB_B2"
                        LPTB-5(900x550)  → beam_id="LPTB-5"
  ✗ NEVER invent an ID you cannot clearly read.
  ✗ Skip ONLY when label is cut off or completely unreadable.

PARSING:
P1 — Strip leading "+":  "+2-T12(C)" → "2-T12(C)"
P2 — Strip ending "+":   "3-T12(A)+" → "3-T12(A)"
P3 — Combined "+" bars — split and copy label to ALL parts:

P5 — Strip these suffixes (keep only the bar part):
     "2-T10 BOTH FACE"       → "2-T10"
     "2-T10(BOTH FACE)"      → "2-T10"
     "2-T10(EXTRA STPS.)"    → "2-T10"
     Remove: BOTH FACE, BOTH FACES, EXTRA STPS, U-BEND
P6 — Order: top bars → extra top → bottom bars → extra bottom → face bars.
P7 — Do NOT include the stirrup bar in reinforcement.
P8 — Ignore words like (CHIPYA) in the final output.
