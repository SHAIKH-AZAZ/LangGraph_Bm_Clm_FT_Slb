===========================
VER. STEEL → reinforcement
===========================

The VER. STEEL cell has bar groups. The bullet symbol • or ○ is just a marker — ignore it.
Each bar group separated by + is a SEPARATE list item.

FORMAT: COUNT-DIAT (replace TOR with T, remove spaces)

Examples from this drawing:
  "• 12-20 TOR"              → ["12-20T"]
  "• 4-25 TOR + • 8-20 TOR"  → ["4-25T", "8-20T"]
  "• 12-20 TOR"              → ["12-20T"]
  "• 16-25 TOR"              → ["16-25T"]
  "• 6-20 TOR + • 6-16 TOR"  → ["6-20T", "6-16T"]
  "• 4-20 TOR + • 8-16 TOR"  → ["4-20T", "8-16T"]
  "• 12-16 TOR"              → ["12-16T"]

CRITICAL: You MUST extract the reinforcement. It is NOT empty. Read VER. STEEL column carefully.

===========================
COL. RING → stirrups
===========================

dia FORMAT: "DIAMETER-TENDCOUNT"
  - DIAMETER = number before TOR (e.g. 8, 10)
  - ENDCOUNT = number after TOR before first @ (e.g. 8)
  - Format: "8-T8", "10-T8"
  - If no number before first @: "T8" or "T10"

spacing FORMAT: all spacing values comma-separated ascending with C/C
  e.g. "100 C/C, 150 C/C"

Examples:
  "8 TOR 8 @ 100 + @ 150 + 8 @ 100 C/C 3-SETS"       → dia="8-T8",  spacing="100 C/C, 150 C/C"
  "10 TOR 8 @ 100 + @ 150 + 8 @ 100 C/C 3-SETS+2LINKS" → dia="10-T8", spacing="100 C/C, 150 C/C"
  "8 TOR @ 100 C/C"                                    → dia="T8",    spacing="100 C/C"

SETS and LINKS are notes — ignore completely.
