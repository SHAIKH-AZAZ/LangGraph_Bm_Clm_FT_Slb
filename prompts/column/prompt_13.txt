=====================================================
DRAWING STRUCTURE
=====================================================

This is a TABLE with:

  BOTTOM ROW   → COLUMN NUMBERS (e.g. GC39, PC174, GC38 ...)
  INNER CELLS  → 3 sub-rows per level × column intersection:

                   STEEL → e.g. "16-T16 + 18-T12"
                   LINKS → e.g. "LOC 1:-T10 -100 AND LOC 2:-T8 -150"

⚠️  CRITICAL: Many cells are EMPTY (blank).

column_no

column_name

The level range. E.g. "BASE TO LG", "GF TO P01", "P06 TO ECO-DECK"

size  (from SIZE sub-row — empty if cell is blank)
-----------------------------------------------------
Format: "300 X 1975 (TYPE-33)"

reinforcement  (from STEEL sub-row — empty if cell is blank)
-----------------------------------------------------
Format: "16-T16 + 18-T12"  or  "4-T16 + 18-T12"  or  "4-T25 + 16-T20"
Split on " + ". Each part is already correct format. Return as list.
  "16-T16 + 18-T12" → ["16-T16", "18-T12"]

stirrups  (from LINKS sub-row — empty if cell is blank)
-----------------------------------------------------
Format: "LOC 1:-T10 -100 AND LOC 2:-T8 -150"

Extract all unique dia values and all unique spacing values:
  dia:     ["T10", "T8"]   (deduplicate if same)

mix
-----------------------------------------------------
Look for a global concrete grade note in the drawing (e.g. "CONCRETE GRADE: M40").

steel_grade
-----------------------------------------------------
